Apa itu HTML?

HTML, atau HyperText Markup Language, adalah bahasa markup standar yang digunakan untuk membuat struktur dan konten halaman web. Dikembangkan oleh Tim Berners-Lee pada tahun 1990-an, HTML memungkinkan pengembang untuk mendefinisikan elemen-elemen seperti heading, paragraf, gambar, tautan, dan banyak lagi menggunakan tag-tag sederhana.

HTML bekerja sebagai kerangka dasar dari sebuah situs web. Tanpa HTML, browser tidak akan tahu bagaimana menampilkan teks, gambar, atau elemen interaktif lainnya. Bahasa ini terus berkembang, dengan versi terbaru seperti HTML5 yang menambahkan fitur-fitur canggih seperti video, audio, dan elemen semantik untuk meningkatkan aksesibilitas dan SEO.

HTML sering digunakan bersama dengan CSS (untuk styling) dan JavaScript (untuk interaktivitas), membentuk trio utama dalam pengembangan web front-end. Memahami HTML adalah langkah pertama yang penting bagi siapa saja yang ingin membangun situs web atau aplikasi web.

Fungsi HTML

HTML memiliki beberapa fungsi diantara lain:


  • Struktur Halaman Web: HTML menyediakan kerangka dasar untuk halaman web, memungkinkan pengembang untuk mengatur konten seperti teks, gambar, dan multimedia dalam format yang terstruktur.
  • Penghubung Hypertext: HTML memungkinkan pembuatan tautan (link) yang menghubungkan satu halaman web dengan halaman lainnya, memfasilitasi navigasi di internet.
  • Integrasi Multimedia: HTML mendukung penyisipan berbagai jenis media seperti gambar, audio, dan video, sehingga membuat halaman web lebih interaktif dan menarik.
  • Formulir dan Input: HTML menyediakan elemen untuk membuat formulir yang memungkinkan pengguna memasukkan data, seperti nama, email, dan komentar.
  • SEO (Search Engine Optimization): Dengan penggunaan tag semantik yang tepat, HTML membantu mesin pencari memahami struktur dan konten halaman web, yang dapat meningkatkan peringkat pencarian.
  • Aksesibilitas: HTML5 memperkenalkan elemen-elemen baru yang meningkatkan aksesibilitas bagi pengguna dengan kebutuhan khusus, seperti pembaca layar.

    Berikut ini merupakan beberapa macam kerangka inti yang membenttuk struktur bahasa markup mulai tag, elemen, dan atribut.
    Tag
    Tag adalah elemen dasar dalam HTML yang digunakan untuk menandai awal dan akhir dari sebuah elemen. Tag biasanya ditulis dalam tanda kurung sudut, misalnya <tagname> untuk membuka tag dan </tagname> untuk menutup tag.
    Elemen
    Elemen adalah kombinasi dari tag pembuka, konten, dan tag penutup. Misalnya, dalam elemen paragraf, tag pembuka adalah <p>, kontennya adalah teks paragraf, dan tag penutupnya adalah </p>. Elemen dapat berisi teks, atribut, atau elemen lainnya.
    Atribut
    Atribut memberikan informasi tambahan tentang elemen HTML. Atribut ditulis di dalam tag pembuka dan biasanya terdiri dari nama atribut dan nilai atribut. Misalnya, dalam tag gambar (<img>), atribut seperti src (sumber gambar) dan alt (teks alternatif) digunakan untuk memberikan informasi tentang gambar tersebut.
